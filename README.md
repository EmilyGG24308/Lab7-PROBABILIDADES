# Laboratorio 7 - MM3014 Teoría de Probabilidades 

## Simulación del Problema del Coleccionista de Cupones

Este laboratorio implementa simulaciones Monte Carlo del clásico **Coupon Collector's Problem** aplicado a un álbum de estampas tipo Panini.

El objetivo es analizar:

- Número esperado de sobres necesarios para completar el álbum.
- Cantidad esperada de estampas repetidas.
- Variabilidad del proceso de colección.
- Probabilidad de completar el álbum con una cantidad fija de sobres.

---

## Tecnologías utilizadas

- Python 3
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Configuración

Instalar las dependencias necesarias:

```bash
pip install numpy matplotlib seaborn notebook
```

---

## Ejecución

Abrir el notebook con:

```bash
jupyter notebook
```

o desde VS Code ejecutando las celdas del archivo `.ipynb`.

---

## Parámetros principales

- Número de estampas: \(N = 100\)
- Estampas por sobre: \(S = 7\)
- Simulaciones Monte Carlo: \(R = 10000\)
